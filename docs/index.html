Installation Guide

Get started with NanoChat in a few simple steps.

1

Chrome Web Store Installation

The simplest way to install NanoChat:

  1. Visit the Chrome Web Store
  2. Search for "NanoChat" in the search bar
  3. Click the "Add to Chrome" button
  4. Confirm the installation when prompted
2

Manual Installation

For developers or advanced users:

  1. Download the latest release from our GitHub Releases page
  2. Unzip the downloaded file to a location on your computer
  3. Open Chrome and navigate to chrome://extensions/
  4. Enable "Developer mode" using the toggle in the top-right corner
  5. Click "Load unpacked" and select the unzipped folder
  6. NanoChat will appear in your extensions list and is ready to use
3

Getting Started

After installation:

  1. Click the NanoChat icon in your Chrome extensions toolbar
  2. The chat interface will open as a sidebar
  3. Grant any requested permissions for optimal functionality
  4. Start asking questions or requesting assistance!
